Dự án: Elearning System
Công nghệ: Next.js + Tailwind CSS + Hero UI

Mô tả: Nền tảng học trực tuyến với giao diện hiện đại, responsive, sử dụng Hero UI để tối ưu trải nghiệm người dùng.

Phân công công việc:

[Trần Cường]: Phát triển trang user (khóa học, quản lý học tập ...).

[Hành Thome]: Phát triển trang admin (quản lý khóa học, danh mục, báo cáo).
